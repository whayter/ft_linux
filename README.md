# ft_linux

Building my own custom Linux system, entirely from source code.

#### => default password: root:ft_linux

#### => extra packages:
* wget
* which
* curl
* ssh
* valgrind
* screen

#### => Cmds

* check kernel version: `cat /proc/version`
* list partitions: `fdisk -l`
